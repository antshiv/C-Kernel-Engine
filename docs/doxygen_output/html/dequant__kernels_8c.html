Detailed Description

Dequantization kernels for GGML-compatible formats.

Implements dequantization from Q4_0, Q4_K, Q6_K, Q8_0 to FP32. These kernels are used as building blocks for quantized GEMM/GEMV.

Key optimization: Dequantize into registers, use immediately in FMA, never write intermediate FP32 values to memory.

Dequantize a single Q4_K block to FP32.

Q4_K uses nested scales: w_fp32 = q * (d * sub_scale) + dmin * sub_min

Where:

  • d, dmin are FP16 super-block values
  • sub_scale, sub_min are 6-bit values packed in scales[12]
  • q is the 4-bit quantized weight (-8 to +7 after offset)
