4 // RMSNorm forward:
5 // For each token t:
6 // r = sqrt( (1/D) * sum_i x_i^2 + eps )
7 // rstd = 1 / r
8 // x_hat_i = x_i * rstd
9 // y_i = gamma_i * x_hat_i
10 //

53 // RMSNorm backward:
54 // Given dY, X, gamma, and cached rstd per token, compute:
55 // dX and dGamma.
56 //
57 // From derivation:
58 // x_hat_i = x_i * rstd_t
59 // m = (1/D) * sum_j (dY_j * gamma_j * x_hat_j)

62 //
