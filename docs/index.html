Overview

C-Kernel-Engine is a high-performance, model-agnostic inference engine for Transformer models, written in pure C with AVX-512 optimizations.

It treats model architecture as data, compiling a configuration file into a specialized C runtime.

Kernel Architecture

The engine uses a three-layer architecture to separate threading logic from pure mathematical kernels.

IR & Codegen Pipeline

Models are defined in JSON, parsed into an Intermediate Representation (IR), and compiled into C code.
